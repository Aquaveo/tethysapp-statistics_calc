Hydrostats App

Welcome to the Hydrostats App! This app is a user interface for the Hydrostats Python package. If you would like to view the Hydrostats functions that correlate with the different tools in this app, links are provided next to each tool in the app. You can use this app for validation of hydrological time series data, as well as the validation of ensemble forecasts and other forecasts.

See the sidebar for the available tools on this app. Also, tutorial videos are provided below.
